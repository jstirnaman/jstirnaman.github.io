Installing Fuseki (with Jena and TDB) on OS X

Notes

Apache Jena binaries downloads

Get Started

  1. Download and extract the Jena binary:
    $ curl -O http://www.apache.org/dist/jena/binaries/jena-fuseki-1.0.0-distribution.tar.gz
    $ tar xvfz jena-fuseki*.gz
  2. Make fuseki-server executable:
    $ cd jena-fuseki-1.0.0 $ chmod +x fuseki-server s-*
  3. Start fuseki-server:
    These options start fuseki-server with an in-memory, non-persistent datastore:
    $ ./fuseki-server --update --mem /ds
